Heart Info System

Aplicativo desenvolvido para identificar o possível agravamento de uma insuficiência cardíaca. Foi criado com base em inteligência artificial e mineração de dados, utilizando um dataset público sobre a doença. O aplicativo foi desenvolvido para Android na plataforma Android Studio, utilizando a linguagem Java e o banco de dados em tempo real Firebase. O aplicativo utiliza o algoritmo K-Nearest Neighbor para a classificação dos dados fornecidos pelo usuário, a fim de verificar um possível agravamento da insuficiência cardíaca. O software foi registrado junto ao INPI (Instituto Nacional da Propriedade Industrial) pelo IFTM. Trata-se de um aplicativo inovador que usa inteligência artificial para auxiliar na descoberta e tratamento da doença.
